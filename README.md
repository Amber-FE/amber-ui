# amber-ui 组件库

### 快速开始

#### 1.安装组件

```
  npm i amber-ui
```
#### 2.引用组件库

```javascript
  // 全部引入
  import 'amber-ui/dist/css/index.css';
  import AmberUi from 'amber-ui';

  Vue.use(AmberUi)

  // 按需引用
  import 'amber-ui/dist/css/picture-stretch.css';
  import { PictureStretch } from 'amber-ui';

  Vue.use(PictureStretch)
```
